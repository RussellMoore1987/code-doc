Grid System

This page documents the Bootstrap 5-style responsive grid utilities included in this project. The system is 12 columns, mobile-first, and based on .container, .row, and .col-* classes.

Container Query Driven

Responsive grid classes are driven by a container query on #content-body, not viewport media queries. This means layouts respond to the content column width (including sidebar states) rather than full browser width.

Grid Anatomy

  • .container* controls horizontal layout bounds and inner padding.
  • .row creates a flex row and manages gutters via CSS variables.
  • .col* classes define column width behavior in a 12-column grid.
  • .g*, .gx*, and .gy* control spacing between columns/rows.
<div class="container">
  <div class="row g-3">
    <div class="col-12 col-md-8">Content</div>
    <div class="col-12 col-md-4">Sidebar</div>
  </div>
</div>

Breakpoints

  • xs default: #content-body width <576px
  • sm: #content-body width >=576px
  • md: #content-body width >=768px
  • lg: #content-body width >=992px
  • xl: #content-body width >=1200px
  • xxl: #content-body width >=1400px

Containers

Use fixed-width containers at breakpoints, or full-width with .container-fluid. The size-specific variants (.container-sm, .container-md, etc.) begin applying a max-width at their named breakpoint. 

<div class="container">...</div>
<div class="container-sm">...</div>
<div class="container-lg">...</div>
<div class="container-fluid">...</div>

Quick Reference

  • Containers: .container, .container-fluid, .container-sm ... .container-xxl
  • Rows: .row
  • Columns: .col, .col-auto, .col-1....col-12, plus -sm/-md/-lg/-xl/-xxl
  • Row-cols: .row-cols-1....row-cols-6 plus responsive variants
  • Offsets: .offset-1....offset-11 plus responsive variants
  • Gutters: .g-0....g-5, .gx-*, .gy-* and responsive gutter variants
Important

This utility set intentionally focuses on the core Bootstrap-like grid API. If you need features like ordering helpers, add matching utility classes to styles.css and document them here.

Tip: compose these grid classes with spacing and flex utilities for fast page layouts.
