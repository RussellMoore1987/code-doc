YouTube Video Systems

Use these components to embed YouTube videos inline, open them in a lightbox, or display a grid of video thumbnails that open in a lightbox gallery with prev/next navigation.

Three usage patterns are available:

  • Inline Embed youtube-embed - a responsive 16:9 iframe that plays directly on the page.
  • Lightbox Trigger youtube-modal - a thumbnail that opens a fullscreen modal player on click.
  • Video Gallery youtube-gallery - a grid of thumbnails sharing a single modal with prev/next navigation.

All three patterns accept the same column modifier classes used by image galleries:

  • 2 Columns img-col-2 for larger tiles and fewer items per row.
  • 3 Columns img-col-3 for a balanced layout.
  • 4 Columns img-col-4 for smaller tiles and more items per row.
  • 6 Columns img-col-6 for compact layouts.
  • 8 Columns img-col-8 for very compact layouts.
  • 12 Columns img-col-12 for extremely compact thumbnail strips.
ℹ YouTube Video IDs

Supply the 11-character YouTube video ID via data-video-id. The system validates this value and rejects anything that doesn't match the expected format, so arbitrary URLs cannot be injected. Videos are embedded using YouTube's privacy-enhanced domain (youtube-nocookie.com).

Inline Embed

Drop a div.youtube-embed with a data-video-id attribute anywhere in the page. The system replaces it with a responsive 16:9 iframe. An optional data-title attribute sets the iframe's accessible title.

<div class="youtube-embed" data-video-id="WhWc3b3KhnY" data-title="Spring - Blender Open Movie"></div>

Lightbox Trigger

A div.youtube-modal or figure.youtube-modal renders as a thumbnail with a play button. Clicking it opens the video in a fullscreen modal overlay. Use data-title for a caption below the player and as the accessible label.

<!-- Plain div trigger -->
<div class="youtube-modal" data-video-id="WhWc3b3KhnY" data-title="Spring - Blender Open Movie"></div>

<!-- Figure trigger with caption -->
<figure class="youtube-modal" data-video-id="WhWc3b3KhnY">
  <figcaption>Caption shown in the modal and beneath the thumbnail.</figcaption>
</figure>

Video Gallery

Wrap multiple [data-video-id] items inside a div.youtube-gallery to get a thumbnail grid. Each tile opens the same shared modal, and the modal's prev/next arrows let the viewer step through all videos in the gallery. Optionally include figure elements with figcaption for per-video captions.

<div class="youtube-gallery img-col-3">
  <div data-video-id="VIDEO_ID_1" data-title="Title 1"></div>
  <div data-video-id="VIDEO_ID_2" data-title="Title 2"></div>
  <figure data-video-id="VIDEO_ID_3">
    <figcaption>Caption for video 3</figcaption>
  </figure>
</div>

Modal Navigation

All lightbox triggers (.youtube-modal) and gallery tiles (.youtube-gallery-item) on the current page share the same modal state. The counter and prev/next arrows let the viewer step through every video on the page in document order, regardless of which component type they came from.

The keyboard shortcuts also work inside the modal:

  • Escape - close the modal
  • ← / → - navigate to the previous/next video

Attribute Reference

Attribute Applies to Description
data-video-id All Required. The 11-character YouTube video ID.
data-title Inline embed, lightbox, gallery items Optional. Sets the iframe accessible title and the modal caption. For galleries, falls back to the figcaption text.
figcaption (child) Figure lightbox, gallery figure items Optional. Caption shown beneath the thumbnail and inside the modal. Overridden by data-title for the modal caption.
img-col-2 … img-col-12 .youtube-gallery Column count modifier, identical to image gallery classes. Responsive breakpoints reduce columns at narrower widths following the same steps as image galleries.
