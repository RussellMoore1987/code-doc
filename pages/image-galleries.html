Image Gallery Systems

Use these gallery containers to build responsive, modal-ready image grids with zero JavaScript. Drop images inside a container with the right classes and the system wires up layout, lazy loading, and the existing modal viewer automatically.

Flex Image Gallery

Flexible, natural-size images laid out with flexbox. Add a column modifier to control how many items appear per row.

Thumbnail Gallery

Small thumbnails that open larger images in the modal viewer. The system tries to locate a larger image using the _big filename pattern and falls back to data-full when provided.

<div class="thumbnail-img-gallery img-col-6">
  <img src="images/pic1.jpg" data-full="images/pic1_big.png" alt="System overview">
  <img src="images/pic2.png" data-full="images/pic2.png" alt="Dashboard">
</div>

Thumbnail Grid

Tip: Use data-full whenever the large image filename does not follow the _big convention.
