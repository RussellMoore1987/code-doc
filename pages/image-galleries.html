Image Gallery Systems

Use these gallery containers to build a responsive, modal-ready image grids. Drop images inside a container with the right classes and the system wires up the layout, it also provides lazy loading.

All Galleries use the same markup structure and can be customized with different column counts.

Here are the available gallery types:

  • Flex Image Gallery flex-img-gallery for natural image sizes and flexible layouts.
  • Uniform Cover Gallery cover-img-gallery for consistent tile sizes with cover cropping.
  • Thumbnail Gallery thumbnail-img-gallery for auto-cropped square thumbnails with large image support.

Here are the available gallery column options:

  • 2 Columns img-col-2 for larger images and fewer items per row.
  • 3 Columns img-col-3 for a balanced layout with moderate image sizes.
  • 4 Columns img-col-4 for smaller images and more items per row.
  • 6 Columns img-col-6 for compact layouts with small images.
  • 8 Columns img-col-8 for very compact layouts with very small images.
  • 12 Columns img-col-12 for extremely compact layouts with tiny images.

Each gallery images can utilize the data-full attribute to specify a larger version of the image for modal or lightbox views. But only the Thumbnail Gallery supports the default image selection behavior, see more here.

Flex Image Gallery

Flexible, natural-size images laid out with a grid. Add a column modifier to control how many items appear per row.

<div class="flex-img-gallery img-col-3">
  <img src="images/pic1.jpg" alt="Sample diagram">
  <img src="images/pic2.png" alt="UI overview">
  <img src="images/pic3.jpg" alt="Workflow sample">
</div>

Uniform Cover Gallery

Uniform tiles with cover-style crops. The gallery converts your images into equal-sized cards with background cover and connects them to the modal viewer. The Uniform Cover Gallery makes equally sized cards with a consistent 4 / 3 aspect ratio.

<div class="cover-img-gallery img-col-4">
  <img src="images/pic1.jpg" alt="Release banner">
  <img src="images/pic2.png" alt="UI overview">
  <img src="images/pic3.jpg" alt="Workflow sample">

Thumbnail Gallery

The thumbnail gallery auto crops the image to a square aspect ratio.

Small thumbnails that open larger images in the modal viewer. The system tries to locate a larger image using the _big filename pattern if the data-full attribute is not provided.

<div class="thumbnail-img-gallery img-col-6">
  <img src="images/pic1.jpg" data-full="images/other_pic.png" alt="System overview">
  <img src="images/pic2.png" alt="Dashboard"> // will get "images/pic2_big.png" automatically
</div>

Thumbnail Grid

Tip: Use data-full whenever the large image filename does not follow the _big convention.
