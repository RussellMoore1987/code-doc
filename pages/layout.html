Sidebar Layout & Resizing

Both sidebars can be collapsed and resized interactively at any time:

  • Collapse — click the arrow icon (‹ / ›) in the sidebar header.
  • Expand — click the arrow icon in the narrow strip that appears when collapsed.
  • Resize — drag the 5 px divider between the sidebar and the center content area. Widths are persisted to localStorage so your preference is remembered across page loads.

Minimum Widths

The left and right sidebars each enforce a minimum width of 220 px. If you drag a sidebar below that threshold it will spring back to the minimum. You can adjust the MIN_SIDEBAR_WIDTH constant in app.js.

Responsive Layout

DevDocs automatically collapses sidebars on narrow viewports to protect the center content area:

  • Below 1200 px — the right sidebar collapses automatically.
  • Below 800 px — the left sidebar also collapses.

On touch devices you can still expand either sidebar manually using the strip expand button. The responsive collapse only triggers once on load; after that, user preference takes over.

Customising breakpoints

Edit the checkResponsiveCollapse() function in app.js and the matching media queries in styles.css to suit your content.
