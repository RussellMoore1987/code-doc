Utility Classes

A reference for all single-purpose utility classes defined in styles.css. These classes can be mixed and matched on any element to handle display, flexbox, spacing, sizing, and text styling without writing custom CSS.

Display

Control the CSS display property of any element.

Class CSS Output
.d-none display: none
.d-inline display: inline
.d-inline-block display: inline-block
.d-block display: block
.d-flex display: flex
.d-inline-flex display: inline-flex
.d-grid display: grid

Example

<!-- Hide an element -->
<div class="d-none">Hidden</div>

<!-- Flex container -->
<div class="d-flex">
  <span>Item 1</span>
  <span>Item 2</span>
</div>

Spacing

Padding and margin utilities follow a consistent scale. The available step values are 0, 5, 10, 15, 20, 24, 30, 40 (px). Each step is available in all directions.

⚠ Tip

Prefer the semantic spacing utilities over inline style attributes to keep markup clean and consistent with the rest of the design.

Padding

Format: .p{direction}-{size}

Prefix Direction Example classes
p All sides .p-0 .p-10 .p-20 .p-40
px Left & Right .px-0 .px-10 .px-24 .px-40

Quick Cheat Sheet

Common combinations you'll reach for most often:

<!-- Centered flex row with gap -->
<div class="d-flex justify-content-center align-items-center gap-3">…</div>

<!-- Space-between header row -->
<div class="flex-sb">
  <span>Left</span>
  <span>Right</span>
</div>

<!-- Horizontally centered block -->
<div class="mx-auto" style="max-width:600px;">…</div>

<!-- Padding shorthand -->
<div class="px-24 py-20">…</div>

<!-- Hide on small screens (toggled via JS) -->
<div class="d-none">Hidden</div>

<!-- Bold muted label -->
<span class="fw-semibold text-muted text-uppercase">Status</span>
