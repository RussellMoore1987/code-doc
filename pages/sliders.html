Sliders

This page demonstrates the use of sliders.

Testimonials

Use a testimonial slider for rotating quotes, feedback, or short text blurbs. This version is intentionally lightweight and uses plain HTML, CSS, and JavaScript.

Optional autoplay for testimonials: data-testimonial-autoplay="true" and data-testimonial-speed="4500" (milliseconds).

Image Slider

Use the image slider for step-by-step screenshots, product tours, or before/after visuals. The slider itself does not force lightbox behavior. If you want lightbox support, add .image-modal to each slide image (or slide figure).

When a lightbox image came from a slider, using the modal "View in page" action will also move the slider to that image and flash-highlight the active slide.

Optional autoplay for image sliders: data-image-slider-autoplay="true" and data-image-slider-speed="4000" (milliseconds).

<section class="image-slider" data-image-slider data-image-slider-autoplay="true" data-image-slider-speed="4000" aria-label="Product screenshots">
    <div class="image-slider-shell">
    <button type="button" class="image-slider-control" data-image-slider-prev aria-label="Previous image">...</button>
    <div class="image-slider-viewport">
      <div class="image-slider-track">
        <article class="image-slide">
            <img src="images/pic1.jpg" class="image-slider-img image-modal" alt="Screenshot one">
        </article>
      </div>
    </div>
    <button type="button" class="image-slider-control" data-image-slider-next aria-label="Next image">...</button>
  </div>
  <div class="image-slider-footer">
    <div class="image-slider-dots" data-image-slider-dots></div>
  </div>
</section>

Continuous Logo Slider

Use this slider for partner, client, or integration logos that should move continuously from right to left. This component is marquee-style (always moving), with optional drag support and optional pause on hover.

Options: data-logo-slider-speed="65" (pixels/second), data-logo-slider-draggable="true", data-logo-slider-stop-on-hover="true". 

<section class="logo-slider" data-logo-slider data-logo-slider-speed="65" data-logo-slider-draggable="true" data-logo-slider-stop-on-hover="true" aria-label="Partner logos">
  <div class="logo-slider-viewport">
    <div class="logo-slider-track">
      <div class="logo-slide"><span class="logo-chip">Rollins</span></div>
      <div class="logo-slide"><span class="logo-chip">Atlas</span></div>
      <div class="logo-slide"><span class="logo-chip">Northstar</span></div>
    </div>
  </div>
</section>
