API Reference

DevDocs exposes a comprehensive JavaScript API for programmatic interaction, plugin development, and advanced customization. This reference documents all public APIs, event systems, and extension points available to developers.

🚀 API Stability

APIs marked as Stable are guaranteed backward-compatible. Beta APIs may change in minor versions. Experimental APIs may change or be removed at any time.

Core API

The core DevDocs API provides fundamental application control and lifecycle management.

Application Object

The global DevDocsApp object exposes core functionality: 

// Access the main application instance
const app = window.DevDocsApp;

// Application properties
app.version         // string: Current DevDocs version
app.state          // object: Current application state
app.config         // object: Configuration settings
app.plugins        // array: Loaded plugin instances

Application Methods

Method Parameters Returns Description Status
navigateTo(pageId) string Promise<void> Navigate to a specific page by ID Stable
loadPage(url) string Promise<HTMLElement> Load page content from URL Stable
getActivePage() none object|null Get current active page info Stable
registerPlugin(plugin) Plugin void Register a new plugin instance Stable
search(query, options) string, object Promise<Array> Execute search query Beta

Application Usage Example

// Navigate to a specific page programmatically
await DevDocsApp.navigateTo('installation');

// Get current page information
const currentPage = DevDocsApp.getActivePage();
console.log('Current page:', currentPage.id, currentPage.title);

// Perform a search
const results = await DevDocsApp.search('configuration', {
  maxResults: 10,
  includeContent: true
});

// Register a custom plugin
DevDocsApp.registerPlugin(new MyCustomPlugin());

Navigation API

The Navigation API controls the left sidebar navigation tree and page routing.

Navigation Methods

Method Parameters Returns Description Status
NavAPI.expandNode(nodeId) string boolean Expand navigation node Stable
NavAPI.collapseNode(nodeId) string boolean Collapse navigation node Stable
NavAPI.getNodePath(pageId) string Array<string> Get navigation path to page Stable
NavAPI.addNode(parentId, node) string, object boolean Add navigation node dynamically Beta
NavAPI.removeNode(nodeId) string boolean Remove navigation node Beta

Navigation Usage Example

// Get the navigation path to current page
const path = NavAPI.getNodePath('deep-topic');
console.log('Navigation path:', path); // ['getting-started', 'first-steps-group', 'step1-group']

// Expand a specific navigation section  
NavAPI.expandNode('advanced');

// Dynamically add a new page to navigation
NavAPI.addNode('reference', {
  id: 'changelog',
  label: 'Changelog', 
  page: 'pages/changelog.html'
});

Search API

The Search API provides programmatic access to search functionality and indexing.

Search Methods

Method Parameters Returns Description Status
SearchAPI.query(text, options) string, object Promise<Array> Execute search query Stable
SearchAPI.rebuildIndex() none Promise<void> Rebuild search index Stable
SearchAPI.addToIndex(item) object void Add item to search index Beta
SearchAPI.getIndexStatus() none object Get indexing status Stable
SearchAPI.clearIndex() none void Clear search index Beta

Search Options

const searchOptions = {
  // Result filtering
  maxResults: 20,              // Maximum number of results
  fuzzyThreshold: 0.6,         // Fuzzy matching sensitivity (0-1)
  includeContent: true,        // Include content snippets in results
  
  // Query processing
  stemming: false,             // Enable word stemming
  stopWords: true,             // Filter common stop words
  caseSensitive: false,        // Case-sensitive matching
  
  // Result ranking
  titleWeight: 2.0,            // Weight for title matches
  headingWeight: 1.5,          // Weight for heading matches  
  contentWeight: 1.0           // Weight for content matches
};

Search Usage Example

// Perform advanced search
const results = await SearchAPI.query('plugin configuration', {
  maxResults: 5,
  fuzzyThreshold: 0.8,
  includeContent: true
});

// Process search results
results.forEach(result => {
  console.log(`${result.title} (score: ${result.score})`);
  console.log(`Preview: ${result.snippet}`);
});

// Check search index status
const status = SearchAPI.getIndexStatus();
console.log(`Index contains ${status.itemCount} items`);

Sidebar API

The Sidebar API controls left and right sidebar behavior, resizing, and state management.

Sidebar Methods

Method Parameters Returns Description Status
SidebarAPI.collapse(side) 'left'|'right' void Collapse specified sidebar Stable
SidebarAPI.expand(side) 'left'|'right' void Expand specified sidebar Stable
SidebarAPI.toggle(side) 'left'|'right' void Toggle sidebar collapsed state Stable
SidebarAPI.setWidth(side, width) string, number void Set sidebar width in pixels Stable
SidebarAPI.getState(side) 'left'|'right' object Get sidebar state info Stable

Sidebar Usage Example

// Collapse the left navigation sidebar
SidebarAPI.collapse('left');

// Set a specific width for the right sidebar
SidebarAPI.setWidth('right', 300);

// Get current state of both sidebars
const leftState = SidebarAPI.getState('left');
const rightState = SidebarAPI.getState('right');

console.log('Left sidebar:', leftState.collapsed ? 'collapsed' : `${leftState.width}px`);
console.log('Right sidebar:', rightState.collapsed ? 'collapsed' : `${rightState.width}px`);

Event System

DevDocs uses a custom event system for communication between components and plugins.

Event Methods

Method Parameters Returns Description Status
EventAPI.on(event, handler) string, function function Register event listener Stable
EventAPI.off(event, handler) string, function boolean Remove event listener Stable
EventAPI.emit(event, data) string, any void Emit event with data Stable
EventAPI.once(event, handler) string, function function Listen for event once only Stable

Built-in Events

  • app:init - Application finished initializing
  • app:ready - All plugins loaded and ready
  • nav:built - Navigation tree rendered
  • page:loading - Started loading new page (pageId)
  • page:loaded - Finished loading page (pageData)
  • page:error - Error loading page (error, pageId)
  • search:query - Search query executed (query, results)
  • search:indexed - Search index built (indexInfo)
  • sidebar:resized - Sidebar width changed (side, width)
  • sidebar:collapsed - Sidebar collapsed/expanded (side, collapsed)
  • theme:changed - Theme toggled (themeName)

Event Usage Example

// Listen for page navigation events
const unsubscribe = EventAPI.on('page:loaded', (pageData) => {
  console.log(`Loaded page: ${pageData.title}`);
  
  // Custom logic after page loads
  initializeCustomFeatures();
});

// Listen for theme changes
EventAPI.on('theme:changed', (themeName) => {
  console.log(`Theme changed to: ${themeName}`);
  updateCustomThemeElements(themeName);
});

// Emit custom events
EventAPI.emit('custom:action', { data: 'example' });

// Remove event listener when needed
unsubscribe();

Plugin API

The Plugin API provides the foundation for creating reusable DevDocs extensions.

Plugin Base Class

class DevDocsPlugin {
  constructor(options = {}) {
    this.name = 'Plugin Name';
    this.version = '1.0.0';
    this.options = { ...this.defaultOptions, ...options };
  }
  
  // Default configuration options
  get defaultOptions() {
    return {
      enabled: true,
      debug: false
    };
  }
  
  // Called when plugin is registered
  init(app) {
    this.app = app;
    this.setupEventListeners();
    this.setupUI();
  }
  
  // Set up event listeners
  setupEventListeners() {
    // Override in subclass
  }
  
  // Set up UI elements
  setupUI() {
    // Override in subclass  
  }
  
  // Called when plugin is disabled
  destroy() {
    this.removeEventListeners();
    this.cleanupUI();
  }
  
  // Remove event listeners
  removeEventListeners() {
    // Override in subclass
  }
  
  // Clean up UI elements
  cleanupUI() {
    // Override in subclass
  }
}

Plugin Lifecycle Hooks

Hook When Called Purpose Status
init(app) Plugin registration Initialize plugin, set up listeners Stable
onPageLoad(pageData) After page loads Process new page content Stable
onSearchIndex(indexData) Search index built Extend search functionality Beta
onThemeChange(theme) Theme toggled Adapt to theme changes Stable
destroy() Plugin disabled Clean up resources Stable

Utility API

DevDocs provides utility functions for common tasks in plugin and theme development.

DOM Utilities

// Element selection and manipulation
UtilAPI.$(selector)              // Enhanced querySelector
UtilAPI.$$(selector)             // Enhanced querySelectorAll  
UtilAPI.createElement(tag, attrs, children) // Create element with attributes
UtilAPI.insertAfter(newEl, target)         // Insert element after target

// CSS class utilities
UtilAPI.addClass(el, className)     // Add CSS class
UtilAPI.removeClass(el, className)  // Remove CSS class
UtilAPI.toggleClass(el, className)  // Toggle CSS class
UtilAPI.hasClass(el, className)     // Check if class exists

Storage Utilities

// LocalStorage wrapper with JSON support
UtilAPI.storage.set(key, value)     // Set value (auto-stringify)
UtilAPI.storage.get(key, default)   // Get value (auto-parse)
UtilAPI.storage.remove(key)         // Remove value
UtilAPI.storage.clear()             // Clear all DevDocs storage

// Session storage
UtilAPI.session.set(key, value)
UtilAPI.session.get(key, default)
UtilAPI.session.remove(key)
UtilAPI.session.clear()

Performance Utilities

// Debouncing and throttling
UtilAPI.debounce(func, delay)       // Debounce function calls
UtilAPI.throttle(func, limit)       // Throttle function calls

// Animation frame utilities
UtilAPI.nextFrame(callback)         // Schedule for next animation frame
UtilAPI.afterPaint(callback)        // Schedule after paint

// Performance measurement
UtilAPI.measureTime(label, func)    // Measure function execution time
UtilAPI.profileStart(label)         // Start performance profile
UtilAPI.profileEnd(label)           // End performance profile
